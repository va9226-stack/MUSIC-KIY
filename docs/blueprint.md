# **App Name**: Harmonic AI

## Core Features:

- AI Music Generation: Generate songs based on user-selected genre tags using the Gemini AI model.
- Music Display with AI curation: Display generated music previews with title and tags. This tool uses AI to pick the most important features and to decide whether to incorporate those into the UI.
- Audio Playback: Stream and play the full generated songs within the app.
- Local Storage: Persist song and genre information locally.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke creativity and depth.
- Background color: Light gray (#F0F2F5) for a clean, modern feel.
- Accent color: Vibrant magenta (#E91E63) to highlight interactive elements.
- Body font: 'Inter' (sans-serif) for clear, readable UI text. Headline font: 'Space Grotesk' (sans-serif) for modern titles and emphasis. Note: currently only Google Fonts are supported.
- Use minimalistic, abstract icons related to music genres and playback controls.
- Implement subtle transitions and animations on music selection and playback events.
